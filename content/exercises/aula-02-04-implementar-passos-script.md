## Enunciado

Implemente um pequeno script que imprima três linhas no formato "Passo N: DESCRICAO": Passo 1: Conectar, Passo 2: Autenticar, Passo 3: Executar. Use apenas `print` e strings.

## Solução

```python
print("Passo 1: Conectar")
print("Passo 2: Autenticar")
print("Passo 3: Executar")
```
