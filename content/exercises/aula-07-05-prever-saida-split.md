## Enunciado

Qual é a saída do programa abaixo?

```python
s = "a b c"
print(s.split(" "))
```

## Solução

A saída é (representação da lista):

```
['a', 'b', 'c']
```

split(" ") quebra nos espaços e retorna lista de strings.
