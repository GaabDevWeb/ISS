## Enunciado

Complete o código para exibir "Total de itens: 42". Use str() para converter o número na concatenação.

```python
total = 42
print("Total de itens: " + _____(total))  # preencha
```

## Solução

```python
total = 42
print("Total de itens: " + str(total))
```
