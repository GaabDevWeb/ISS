## Enunciado

Uma startup quer que todos os scripts de onboarding comecem com uma linha comentada explicando o propósito do arquivo e, em seguida, exibam uma mensagem de boas-vindas no terminal. Isso padroniza a experiência para novos desenvolvedores.

**Tarefa:** No mesmo programa, use `#` para escrever um comentário de uma linha com o texto: "Script de boas-vindas ao time." Depois use `print()` para exibir a mensagem: "Bem-vindo ao time de desenvolvimento!".

## Solução

```python
# Script de boas-vindas ao time.
print("Bem-vindo ao time de desenvolvimento!")
```
