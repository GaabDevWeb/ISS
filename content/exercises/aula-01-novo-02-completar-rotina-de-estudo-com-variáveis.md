## Enunciado

Complete o código para exibir a frase `Hoje vou estudar 45 minutos de Python`.

```python
minutos = 45
linguagem = "Python"
# complete abaixo
print(______________)
```

## Solução

```python

linguagem = "Python"
print(f"Hoje vou estudar {minutos} minutos de {linguagem}")
```
