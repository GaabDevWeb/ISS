## Enunciado

O setor de RH de uma escola de cursos livres deseja preparar o sistema para o registro de novos alunos. O coordenador pedagógico solicitou que os dados sejam organizados em variáveis separadas antes de qualquer processamento futuro. O cenário envolve o armazenamento do nome de uma disciplina e o código da turma. O impacto esperado é a organização estruturada das informações acadêmicas.

**Tarefa:** Declare uma variável chamada `disciplina` com o valor `"Python"` e uma variável chamada `turma` com o valor `102`. Utilize dois comandos `print()` separados para exibir o conteúdo de cada variável no terminal.

## Solução

```python
disciplina = "Python"
turma = 102
print(disciplina)
print(turma)
```
