## Enunciado

Complete a string para exibir: O arquivo do usuário está em 'entrada.txt'. Use aspas duplas externas para que as aspas simples internas não fechem a string.

```python
msg = _____  # preencha com a mensagem
print(msg)
```

## Solução

```python
msg = "O arquivo do usuário está em 'entrada.txt'."
print(msg)
```
