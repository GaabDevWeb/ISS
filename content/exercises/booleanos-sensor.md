## Enunciado

Uma empresa de segurança patrimonial precisa de um verificador para o estado dos sensores de um armazém. O cenário envolve apenas dois estados possíveis: ativo ou inativo. O programa deve guardar o estado atual do monitoramento para que outros sistemas possam ler essa informação lógica.

**Tarefa:** Crie uma variável chamada `sensor_ativo` e atribua a ela o valor booleano `True`. Em seguida, exiba o valor da variável no terminal utilizando `print()`. Altere seu valor para `False` e imprima novamente.

## Solução

```python
sensor_ativo = True
print(sensor_ativo)
sensor_ativo = False
print(sensor_ativo)
```
