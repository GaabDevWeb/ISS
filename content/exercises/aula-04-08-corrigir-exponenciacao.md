## Enunciado

O código deveria calcular 2 elevado a 3 (8). Corrija: use o operador de exponenciação `**`.

```python
resultado = 2 * 3
print(resultado)
```

## Solução

```python
resultado = 2 ** 3
print(resultado)
```
