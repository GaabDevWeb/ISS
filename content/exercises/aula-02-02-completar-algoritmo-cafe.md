## Enunciado

Complete o algoritmo abaixo com um passo que falta. O programa deve imprimir quatro linhas: "Aquecer água", "Colocar café no filtro", "Despejar água", "Servir".

```python
print("Aquecer água")
print("Colocar café no filtro")
print(_____)  # preencha: despejar água
print("Servir")
```

## Solução

```python
print("Aquecer água")
print("Colocar café no filtro")
print("Despejar água")
print("Servir")
```
