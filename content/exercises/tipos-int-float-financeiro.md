## Enunciado

Um analista financeiro está criando um controle de contratos para uma microempresa. O cenário exige a distinção clara entre números inteiros (contagem de contratos) e números decimais (valores monetários) para garantir precisão contábil. O programa deve armazenar esses dados e confirmar os tipos de dados processados.

**Tarefa:** Crie as variáveis `contratos = 15` e `faturamento = 3200.75`. Use a função `print()` para exibir o tipo da variável `contratos` e o tipo da variável `faturamento` utilizando a função `type()`.

## Solução

```python
contratos = 15
faturamento = 3200.75
print(type(contratos))
print(type(faturamento))
```
