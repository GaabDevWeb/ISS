## Enunciado

Complete o código para exibir "ERRO: timeout". Use o operador + para concatenar.

```python
tipo = "ERRO"
msg = "timeout"
print(_____ + ": " + _____)  # preencha
```

## Solução

```python
tipo = "ERRO"
msg = "timeout"
print(tipo + ": " + msg)
```
