## Enunciado

O código abaixo tem três passos. Estenda-o para incluir um quarto passo: "Registrar log", mantendo a ordem. A saída deve ter quatro linhas.

```python
print("Validar entrada")
print("Processar")
print("Gerar saída")
```

## Solução

```python
print("Validar entrada")
print("Processar")
print("Gerar saída")
print("Registrar log")
```
