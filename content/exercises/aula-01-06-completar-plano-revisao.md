## Enunciado

Complete o código para exibir uma única linha no formato: "Revisar tópico: NOME_TOPICO em OFFSET dias." Use as variáveis já declaradas.

```python
nome_topico = "Loops"
offset_dias = 2
print("Revisar tópico:", nome_topico, "em", _____, "dias")  # preencha
```

## Solução

```python
nome_topico = "Loops"
offset_dias = 2
print("Revisar tópico:", nome_topico, "em", offset_dias, "dias")
```
