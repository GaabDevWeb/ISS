## Enunciado

O departamento de marketing de uma rede de cinemas quer testar a interface de seus novos totens de autoatendimento. Para isso, o setor de desenvolvimento precisa de um script simples que valide a saída de texto padrão do sistema para garantir que o hardware está processando comandos corretamente. O programa deve exibir a saudação oficial solicitada pela gerência na tela do terminal.

**Tarefa:** Escreva um programa que utilize a função `print()` para exibir a mensagem: "Olá Mundo! Bem-vindo ao Cinema Digital".

## Solução

```python
print("Olá Mundo! Bem-vindo ao Cinema Digital")
```
