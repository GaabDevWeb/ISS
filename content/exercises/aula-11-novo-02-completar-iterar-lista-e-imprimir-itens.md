## Enunciado

Complete para imprimir cada item da lista em uma linha.

```python
itens = ["a", "b", "c"]
for x in ____________:
    print(x)
```

## Solução

```python
itens = ["a", "b", "c"]
for x in itens:
    print(x)
```
