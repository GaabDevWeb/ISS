## Enunciado

Qual é a saída do programa abaixo?

```python
topico = "Python"
horas = 2 * 3
print("Tópico:", topico)
print("Horas esta semana:", horas)
```

## Solução

A saída é:

```
Tópico: Python
Horas esta semana: 6
```
