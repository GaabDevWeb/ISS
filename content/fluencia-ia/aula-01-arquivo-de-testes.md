## Propósito deste arquivo

Este arquivo existe como **conteúdo de teste** para validar:

- carregamento de uma disciplina nova via `disciplines.json`
- roteamento para aula via `lessons.json`
- renderização de frontmatter e exercícios no `aula.html`
